убрать папки или файлы от отслеживания гитом:
git rm -r --cached .idea //убирает отслеживание папки .idea
посмотреть настройки гита 
git config --list
установка имени и почты используемых гитом
git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

git clone https://....git
после этой команды с репозитория все будет скачано на локальную машину в папку DataTransfer откуда запущен гит,
если мы добавили файл в локальную папку DataTransfer, например test.txt мы должны его добавить в проект
для этого переходим в папку DataTransfer и и набираем
git add test.jar
чтобы добавить сразу все новые файлы в проект набираем
git -A
чтобы увидеть статус нашего проекта набираем
git status
увидим что файл добавлен в наш проект (вместо красного цвета станет зеленым),
затем нужно сделать соммит
git commit -m "add file test.jar"
затем чтобы закачать все изменения в репозиторий нужно 
git push
вводим имя пользователя и пароль, которые были зарегины на github
чтобы принять все изменения с репозитория нужно выполнить
git pull
если мы выполнили 
git commit 
и больше ничего не написали, то мы попадаем в окно из которого можно выбраться 
нажав ESC затем :wq и нажать enter
