Горячие клавиши:

Поиск:
Ctrl+N поиск классов в проекте
Ctrl+Shift+N поиск по всем файлам которые есть в проекте
Ctrl+Shift+F расширенный поиск текста в проекте
Ctrl+Alt+Shift+N является быстрым способом нахождения элемента в вашем проекте. 
Ctrl+Q получить документацию по конкретному элементу кода, для чего нужно установить на нем курсор и нажать комбинацию Ctrl+Q
Ctrl+B переход к описанию класса
Ctrl+F12 просмотр всех методов в классе
Ctrl+Alt+F7 просмотр где используется переменная клас и тп
Ctrl+E открыть недавно используемые файлы
Ctrl+p просмотр принимаемых методом, конструктором переменных
Alt+F7 показывает где используется в проекте метод класс и т п 
Ctrl+Shift+I показ определения переменной, класса во всплывающем окне

Git:
Ctrl+Alt+A добавить файл или папку в гит
Ctrl+K коммитить изменения
Ctrl+Shift+K запушить изменения
Ctrl+T подтянуть изменения с удаленного репозитория

Настройки:
Ctrl+Alt+S выводит окно настроек
Ctrl+Shift+Alt+S  настройка версии JDK для текущего проекта
Ctrl+Shift+A быстрый поиска по настройкам, можно поискать Zoom и поставить галочку чтобы скролом текст увеличивался уменьшался

Форматирование:
Ctrl+ Shift + / закоментировать блок кода
Ctrl + / закомментировать строку
Alt+Enter нажимаем когда нужно импортировать класс
Ctrl+Shift+Enter при написании метода или условия добавляет автоматически фигурные скобки
Ctrl+Alt+L,Ctrl+Alt+I автоформатирование кода(предварительно нужно выделить участок кода)
Shift+F6 переименовывание автоматическое
Ctrl+D дублирует строчку в которой находится курсор
Ctrl+Alt+T можно генерировать вокруг выделенного кода try/catch, if/else, while и т.д.
Alt+Insert может генерировать конструкторы к выделенным полям класса, геттеры и сеттеры

Отладка:
F9 при отладке шагаем по breake точкам 
F8 при отладке шагает по строчкам кода
при отладке кода выделяем переменную и нажимаем Ctrl+F1 чтобы вызвать меню которое покажет значение этой переменной

Разное:
CTRL+Alt+стрелка влево вернуться на шаг назад, стрелка вправо на шаг вперед
Ctrl + Shift + +/- разворачивает/сворачивает методы 
Ctrl+Shift+Blank — вызов smart comlition — фильтрует список из методов и переменных ожидаемого типа.
1) если написать sout и нажать tab, то получится System.out.println(), 
если написать psvm и нажать tab, то получиться public static void main(String[] args) {}
Shift+клик лкм - принаведении на вкладку закрывает ее, в тексте при зажатом Shift выделяет код от курсора до нового места курсора
Нажимая два раза Alt и удерживая его показывается панель инструментов

посмотреть историю изменений кода в любом файле ПКМ -> Local History -> Show History
